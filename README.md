# 一个可以离线使用的PC浏览器主页
> 原版是以移动为主的H5仿夸克浏览器主页 https://github.com/liumingye/quarkHomePage  
> 改版以PC为主，做为一个可本地化离线使用的直观便捷的浏览器主页  
> **如下载后放到D盘运行index.html并设置为浏览器主页即可**  


## 暂时
* 反人类布局，假如你也看腻了居中
* 左侧部分搜索结果右侧框架/ajax执行
* 右侧部分工具/api在左侧或右侧框架/ajax执行
* 用颜色或者图标区分（如某些工具在框架内打开更为便捷）
